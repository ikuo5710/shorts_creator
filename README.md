# Shorts Creator

This project automatically generates short videos with audio based on a given prompt.

## Usage

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Set up environment variables:**
   - Create a `.env` file from the `config/.env.example` file and add your API keys.

3. **Run the pipeline:**
   ```bash
   python src/run_pipeline.py "your prompt here"
   ```
